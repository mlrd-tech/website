    User#

    Presuming:

    • User: mlrd
    • Password: ...
    • Database: mlrd

    Use caching_sha2_password and grant full access only to the database that mlrd uses:

    CREATE USER IF NOT EXISTS mlrd IDENTIFIED WITH caching_sha2_password BY '...';

GRANT ALL ON mlrd.* TO mlrd@'%';
